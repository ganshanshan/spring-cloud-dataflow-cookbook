# HTTP to MySQL

![image](https://cloud.githubusercontent.com/assets/106908/20025231/c4f5b7a0-a32f-11e6-9f47-288dd387b432.png)

## Deploy stream

``` shell
cf create-service p-mysql 512mb name-db
```

``` shell
../deploy-stream.sh http-to-mysql.log
```
